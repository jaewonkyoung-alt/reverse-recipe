WEEK 3

프론트엔드 개발 — 3가지 핵심 목표

사용자가 직접 만지는 화면과 데이터 흐름을 완성한다

GOAL 01
🖥️
React UI 화면 구현
사용자가 실제로 사용할 5개 화면을 React로 개발한다. 홈·냉장고·레시피 추천·장보기·마이페이지로 구성되며, 모바일에서도 자연스럽게 작동하는 반응형 레이아웃을 적용한다.
React Tailwind CSS 모바일 최적화
GOAL 02
🔗
프론트 ↔ 백엔드 API 연동
2주차에 만든 REST API와 화면을 실제로 연결한다. 로그인 시 JWT 토큰을 저장하고, 이후 냉장고 재료 조회·추가·삭제가 DB에 즉시 반영되는 실시간 데이터 흐름을 구현한다.
JWT 인증 Axios Zustand 상태관리
GOAL 03
🔍
식재료 자동완성 DB 구축
교수님 피드백(등록 번거로움)을 반영하여, 가정에서 자주 쓰는 식재료 230개를 DB화했다. 2~3글자만 입력하면 즉시 후보가 뜨고 소비기한도 자동으로 입력되어 등록 과정을 대폭 단축한다.
230개 식재료 자동완성 소비기한 자동설정
